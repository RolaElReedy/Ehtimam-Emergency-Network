# Ehtimam-Emergency-Network
Real-time emergency support circle application built with Qt and Boost.Asio.

# Ehtimam – Emergency Support Circle Network

# Description
Ehtimam is a real-time emergency communication system that allows users to instantly notify a trusted circle during urgent situations.

# Technologies
- Qt (GUI)
- Boost.Asio (Networking)
- JSON (Structured Messaging)
- GoogleTest & GoogleMock (Unit Testing)
- CMake (Build System)

## Architecture
Client-Server asynchronous TCP system with structured JSON protocol.

## Authors
- Rola ElReedy

## Features Implemented (Milestone 2)
- Client-server communication (Boost.Asio)
- Emergency alert system
- Real-time broadcasting
- Basic circle management (add/remove/list users)
- Status updates (text messages)

## How to Run
1. Run server:
   ./server_app
2. Run two clients:
   ./client_app
